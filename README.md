# n8n-crm-telegram-bot

CRM automatizado com IA usando N8N + Telegram + Google sheets

# 🤖 CRM Automatizado com IA

## 🚀 Sobre o projeto
Sistema de atendimento automatizado com inteligência artificial integrado ao Telegram.

## ⚙️ Funcionalidades
- Atendimento automático
- Classificação de intenção com IA
- Listagem de produtos
- Criação de pedidos
- Registro em Google Sheets (CRM)

## 🛠️ Tecnologias
- n8n
- Telegram Bot
- Google Sheets
- IA (Gemini/OpenAI)

## 💼 Objetivo
Automatizar vendas e atendimento para pequenos negócios.
